🎯 Propósito del Procedimiento

Configurar y resolver problemas de acceso remoto a los sistemas Capstone mediante VPN (Azure VPN Gateway, FortiClient) y conexiones de escritorio remoto (RDS) para acceso a SAP y otros sistemas internos.

📋 Alcance

✓ Incluye

  • Instalación y configuración de cliente FortiClient VPN
  • Configuración de Azure VPN en equipos corporativos
  • Configuración de conexión RDS para acceso remoto a SAP
  • Diagnóstico de problemas de conexión VPN

✗ No incluye

  • Problemas de internet del usuario (no es VPN si el usuario no tiene internet)
  • Falla del VPN Gateway en Azure → N3 Networking
  • Acceso Bastion → SOP-NET-004
✅ Requisitos Previos
  • Instalador de FortiClient o configuración de Azure VPN
  • Credenciales AD activas y MFA configurado
  • Cuenta con permiso de acceso VPN
📄 Procedimiento Paso a Paso
Paso 01
Verificar que el problema es de VPN y no de internet

Confirmar que el usuario tiene acceso a internet antes de diagnosticar VPN. Si no tiene internet: derivar a su proveedor de internet (si es en casa) o a SOP-NET-001 (si es en sitio). Si tiene internet pero no VPN: continuar diagnóstico.

⚠ Nota: El 20% de los tickets de VPN son realmente problemas de internet del usuario. Verificar siempre primero.
Paso 02
Identificar el tipo de VPN requerida

Determinar qué VPN usa el usuario: (a) FortiClient VPN: para acceso a recursos internos de Capstone desde equipos corporativos. (b) Azure VPN: para conexiones de sitio a sitio o usuarios específicos en Azure. (c) RDS SAP: para acceso a SAP desde fuera de la red corporativa. Cada tipo tiene configuración diferente.

Paso 03A
[FortiClient] Instalar y configurar

Descargar FortiClient desde el repositorio interno o portal de TI. Instalar con permisos de administrador. Configurar el servidor VPN: dirección IP o DNS del Gateway FortiGate de Capstone. Configurar usuario y password (credenciales AD). Verificar conexión. Si hay error de certificado: importar el certificado corporativo.

⚠ Nota: El servidor FortiGate Gateway lo proporciona Networking_Support. Si no se conoce la dirección actual, solicitarla a N3.
Paso 03B
[RDS SAP] Configurar conexión de escritorio remoto

Configurar en Windows: Conexión a Escritorio Remoto (mstsc.exe). Ingresar la dirección del servidor RDS SAP (proporcionada por SAP-Chile_Tech). Conectar con las credenciales del dominio Capstone. Verificar que SAP carga correctamente.

⚠ Nota: Si el RDS SAP no conecta habiendo VPN activa: puede ser que la cuenta no tenga permiso en el servidor RDS. Escalar a SAP-Chile_Tech para verificar acceso RDS.
Paso 04
Verificar y cerrar

Confirmar que el usuario puede conectarse a los recursos internos necesarios a través de la VPN. Para RDS SAP: confirmar que SAP carga correctamente. Cerrar con 'Resolved'.

📡 Criterios de Escalamiento
  • VPN Gateway de Azure o FortiGate no disponible → N3 Networking_Support_Chile_Tech
  • Usuario sin permiso de acceso VPN → ADC debe autorizar
  • Certificados VPN expirados → N3
